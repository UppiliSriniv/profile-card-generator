# User Profile Card Generator

A Full Stack Flask Web Application that allows users to create attractive profile cards by entering personal details.

## Features

- Flask Backend
- POST Request Handling
- Dynamic Profile Generation
- Responsive Design
- Skill Tags
- Profile Image Support
- Modern UI

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja2

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run application

python app.py

3. Open browser

http://127.0.0.1:5000
